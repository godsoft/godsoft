모바일 Vibrator 디바이스 API 가이드

설명

행정안전부 전자정부 표준프레임워크 센터에서는 전자정부 표준프레임워크를 이용하여 하이브리드 앱 개발 을 할 수 있는 모바일 디바이스 API 실행환경, 개발환경, 가이드프로그램을 제작 배포하고 있습니다.

VibratorAPI 가이드 프로그램은 모바일 디바이스 API 실행환경을 활용하여 하이브리드 앱을 개발하시는 분들이 구현 시 참고 및 활용될 수 있도록 Vibrator API에 대한 가이드 프로그램 입니다.

기능사양

  • 적용 오픈 소스 : PhoneGap cordova1.9.0, jQuery Mobile 1.1.1, jQuery 1.7.1, eGovMoible 1.0, iScroll4
  • 적용 라이브러리 : android.jar, annotation.jar, spring-android-rest-template-1.0.0.RELEASE.jar, spring-android-core-1.0.0.RELEASE.jar
  • 운영 플랫폼 : Android SDK Revision 18(적용빌드번호 : GINGERBREAD EL21), 전자정부표준프레임워크 2.5
  • 개발환경 : 전자정부표준프레임워크 개발환경2.5
  • 적용 API :
    1. PhoneGap : Notification Device API
    2. 전자정부표준프레임워크 : Interface Device API

기능설명

  • 소리로 알림 : 알림 설정 시간이 되면 Beep를 사용하여 사용자에게 알립니다.
  • 진동으로 알림 : 알림 설정 시간이 되면 Vibrator를 사용하여 사용자에게 알립니다.
  • 알림 목록 보기 : 서버에 저장된 알림 설정 정보를 화면에 표출하여 보입니다.
